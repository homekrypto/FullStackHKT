Setup Instructions

  1. Extract the tar.gz file
  2. Run npm install to install dependencies
  3. Create .env file with your DATABASE_URL
  4. Run npm run db:push to set up the database
  5. Run npm run dev to start development server

What's Included

  • Frontend: Complete React app with 23 pages
  • Backend: Express.js API with authentication
  • Database: PostgreSQL schema with Drizzle ORM
  • Features: Investment tracking, wallet integration, blog system
  • Styling: Tailwind CSS with dark/light themes
